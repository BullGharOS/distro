Hitta den bästa mjukvaran

Säg adjö till att söka på webben efter nya program. Det finns ett överflöd av program tillgängliga för Ubuntu MATE, och vissa kan finns utbudet överväldigande. Programbutiken är ett omsorgsfullt bearbetat urval av program som är bäst på vad de gör för att komplettera Ubuntu MATE. Erfarna användare kan enkelt köa alla deras favoritprogram för installation och utföra detta i en och samma åtgärd.
